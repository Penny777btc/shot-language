shot-language · 安装与使用说明
================================

这是什么
--------
一个给「视频剪辑 / 录屏 / 出镜」用的画面构图判断技能（Claude skill）。
它把摄影构图原理，翻译成「画面好不好、该怎么改」的可执行判断 + 质检清单。
完整说明见 README.md（可用浏览器或任意文本编辑器打开），
里面有 8 个功能点的「使用前 / 使用后」对比图（在 images 文件夹）。


用法一：只想学构图，不用 AI
---------------------------
直接看 README.md 和 images 里的对比图即可，无需安装。
SKILL.md 末尾有「八条构图错误」自检清单，可拿去给自己的成片做体检。


用法二：当 Claude Code 技能来用
-------------------------------
1. 解压本压缩包，得到 shot-language 文件夹。
2. 把整个 shot-language 文件夹放到下面任一位置：

   · 只在某个项目里用：
       你的项目目录/.claude/skills/shot-language/

   · 全局都能用：
       macOS / Linux :  ~/.claude/skills/shot-language/
       Windows       :  C:\Users\你的用户名\.claude\skills\shot-language\

3. 重新打开 Claude Code。之后你做「选片定景别 / 运镜 / 出镜机位 /
   成片质检 / 画面为什么不好看」这类事时，它会自动参考这个技能；
   你也可以直接对它说：

       "用 shot-language 看看这段画面有什么问题"
